    Canvas Deployment README

    Save this file as business_math_quest_randomized_points_v4.html, upload it to Canvas Files, publish it, and embed it in a Canvas Page with an iframe that points to the uploaded file URL.

    This game is designed as a practice/remediation tool. It saves progress in the learner's browser localStorage and does not automatically write scores to the Canvas Gradebook.

    For official grading, use a Canvas quiz or assignment, or implement a formal LTI/API integration separately.
